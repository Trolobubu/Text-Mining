# Text Mining Project — Professional Mentions Classification

This project explores text classification techniques to identify and analyze mentions of professional roles in social media posts.  
It was developed as part of an NLP exercise using the **ProfNER dataset**, simplified for educational purposes.

## Overview
The notebook implements preprocessing, model fine-tuning, and evaluation using **HuggingFace Transformers**.  
The main goal is to detect whether a tweet mentions a profession (binary classification).

## Technologies Used
- Python
- Transformers (HuggingFace)
- PyTorch
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
